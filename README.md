Output Description – StudyTrack App

The StudyTrack application is a simple and user-friendly student task manager developed using Flutter. The app allows users to efficiently manage their daily tasks in an organized manner.

When the application is launched, the user is presented with a clean interface displaying a list of tasks. Users can enter a new task using the input field and add it to the list by clicking the “Add Task” button. Each task appears in the list view format.

The app provides functionality to mark tasks as completed using a checkbox. Once marked, the task visually indicates completion, helping users track their progress. Additionally, users can remove tasks from the list using the delete (🗑) button provided alongside each task.

All tasks are displayed dynamically in a scrollable list, ensuring smooth handling even when multiple tasks are added. The application focuses on simplicity, usability, and efficient task tracking for students.

Overall, the StudyTrack app successfully demonstrates basic Flutter concepts such as widget structure, state management, and user interaction while providing a practical solution for daily task management.
